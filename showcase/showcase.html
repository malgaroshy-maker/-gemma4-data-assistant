Gemma 4 Data Assistant

A local-first, offline AI data analyst powered by Google's Gemma 4 E4B model.

Python 3.10+ Streamlit Gemma 4 E4B llama.cpp MIT

🎙️ Offline Voice-to-Text

Native Gemma 4 ASR. 100% local — no internet required.

📊 AI-Generated Charts

Dynamic Matplotlib/Seaborn charts via agentic tool execution.

💾 Excel Report Generation

AI builds formatted Excel workbooks with tables, charts, and summaries.

🧠 Deep Reasoning

Watch the AI think step-by-step with transparent reasoning tabs.

🌐 Bilingual EN/AR with RTL

Full Arabic UI, right-to-left layout, Arabic chart text.

👁️ Vision / Image Analysis

Upload screenshots, compare charts, analyze images with AI.

⚡ Context Management

128K context, auto-trim, real-time usage meter with warnings.

🚀 Hardware-Optimized

Flash Attention, KV q4_0, speculative decoding for RTX 4060 8GB.

🚀 Quick Start

# 1. Start server
./llama-opencode.bat

# 2. Launch app
.venv\Scripts\Activate.ps1
streamlit run app.py

Then open http://localhost:8501, upload a file, and start chatting.
